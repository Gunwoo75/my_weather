OpenWeatherMap API를 사용하여 현재 날씨, 시간별/단기 예보, 그리고 현재 기온에 맞는 옷차림 추천 및 포켓몬고 날씨 부스트 정보를 제공하는 반응형 웹 애플리케이션입니다.
수원대학교 수업에서 진행하는 프로젝트 입니다. 바이브 코딩으로 작성된 코드입니다. 
Netlify를 통해 배포 되고 있습니다. https://mywheather-suwon-univ.netlify.app/
처음 접속시 현재 위치 기반으로 날씨 정보를 검색하기 때문에 위치 접근을 허용해야 합니다.

-----------------------------------------------------------------------------------------------
-- 주요 기능 --
실시간 날씨 및 예보: 도시 이름 검색을 통해 현재 온도, 습도, 풍속 및 최대 5일간의 예보를 표시합니다.

모바일 퍼스트 반응형 디자인: 화면 크기에 따라 메인 콘텐츠와 확장 기능(옷차림/부스트)의 배치가 유연하게 변경됩니다. 

단위 토글: 섭씨(°C)와 화씨(°F)를 손쉽게 전환할 수 있습니다.

최근 검색 기록: localStorage를 사용하여 최근 검색했던 도시를 저장하고 빠르게 재검색할 수 있습니다.

-- 추가 기능 --
옷차림 추천: 현재 기온 및 강수 예보에 따른 최적의 옷차림 정보를 제공합니다.

포켓몬고 날씨 부스트: 현재 날씨에 따른 게임 내 타입 부스트 정보를 제공합니다.
